# powerbi-sales-analysis
E-Commerce Analytics Dashboard (Power BI)

Projektbeschreibung
Dieses Projekt ist ein interaktives Business-Intelligence-Dashboard, das auf dem brasilianischen E-Commerce-Datensatz Olist basiert.
Ziel des Projekts ist es, den gesamten Geschäftsprozess — von der Bestellung über die Logistik und Zahlungen bis hin zum Kundenfeedback — datenbasiert zu analysieren und zu visualisieren.

Der Fokus liegt auf:
-- Umsatzanalyse
-- Lieferperformance
-- Zahlungsstruktur
-- Kundenzufriedenheit

Das Dashboard ist in vier analytische Bereiche gegliedert: Sales, Logistics, Payments und Feedback.

Inhalt der Dashboards
  Sales
Analyse von Umsatz, Bestellvolumen und Produktperformance:
-- Gesamtumsatz und Anzahl der Bestellungen
-- Durchschnittlicher Warenkorbwert
-- Umsatz nach Kategorien, Produkten und Bundesstaaten
-- Zeitliche Entwicklung der Verkäufe

  Logistics
Bewertung der Lieferqualität:
-- Durchschnittliche Lieferzeit
-- Anteil pünktlich gelieferter Bestellungen
-- Full-Cycle-Time (Bestellung → Lieferung)
-- Vergleich der Logistikleistung nach Bundesstaaten und Kategorien

  Payments
Analyse des Kunden-Zahlungsverhaltens:
-- Verteilung der Zahlungsmethoden
-- Durchschnittlicher Bestellwert pro Zahlungsmethode
-- Nutzung von Ratenzahlungen (Installments)
-- Entwicklung der Zahlungswerte über die Zeit

  Feedback
Analyse der Kundenzufriedenheit:
-- Anzahl der Bewertungen
-- Durchschnittliches Rating
-- Anteil positiver Bewertungen
-- Reaktionszeit auf Kundenfeedback
-- Zusammenhang zwischen Lieferzeit und Bewertung
